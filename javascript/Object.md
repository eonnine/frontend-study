# 네이티브 객체 vs 호스트 객체

## Javascript의 Object 종류
- Native Object  
- Host Object  
- User Defined Object  

<br>

## 네이티브 객체 (Native Object)

**ECMAScript 명세에 정의된 객체**이다.

* `Object`
* `Function`
* `Date`
* `Math`
* `parseInt`
* `eval` ...

<br>

## 호스트 객체 (Host Object)

**자바스크립트를 실행하는 환경에 종속된 객체와 사용자가 정의한 객체**이다.

* `window`
* `document`
* `location`
* `XMLHttpRequest`
* `querySelectorAll` ...

<br>
